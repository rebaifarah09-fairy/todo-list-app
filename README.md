# 📝 To-Do List Web App

Application web simple et moderne pour gérer vos tâches quotidiennes ✔️
Développée avec Flask et Python.

![Python](https://img.shields.io/badge/Python-3.10-blue)
![Flask](https://img.shields.io/badge/Flask-WebApp-black)
![SQLite](https://img.shields.io/badge/SQLite-Database-lightgrey)

---

## 📋 Description du projet

Cette application permet aux utilisateurs de gérer efficacement leurs tâches quotidiennes.

Fonctionnalités principales :

* Ajout de tâches
* Modification de tâches
* Suppression de tâches
* Marquage comme terminé
* Sauvegarde des données avec SQLite

## ⚙️ Installation

1. Cloner le projet :

```
git clone https://github.com/votre-username/todo-app.git
cd todo-app
```

2. Créer un environnement virtuel (optionnel) :

```
python -m venv venv
source venv/bin/activate   # Linux / Mac
venv\Scripts\activate      # Windows
```

3. Installer les dépendances :

```
pip install -r requirements.txt
```

---

## ▶️ Lancer l'application

```
python app.py
```

Puis ouvrir dans votre navigateur :

```
http://127.0.0.1:5000
```
